Style Menu

Copy Font: copies the font name, style and size to the pasteboard, so that it can be ‘pasted’ onto other selected text.

Paste Font: pastes a font name, style and size from the pasteboard onto selected text.

Copy Ruler: copies the paragraph style at the insertion point to the pasteboard, so that it can be ‘pasted’ onto other selected paragraphs.

Paste Ruler: pastes a paragraph style from the pasteboard onto the selected paragraphs.

Copy Font/Ruler: copies the font family, style, and size as well as the paragraph style at the insertion point to the pasteboard.

Paste Font/Ruler: pastes a font family, style and size as well as a paragraph style from the pasteboard onto selected text.

Select by…

Font Style: selects all text matching the style of the font at the insertion point. Example: ‘Arial Bold Italic.’

Font Family: selects all text matching the font family at the insertion point. Example: ‘Arial.’

Font Size: selects all text matching the size of the font at the insertion point.

Font Color: selects all text matching the color of the font at the insertion point.

Highlight Color: selects all text matching the highlight color of the text at the insertion point.

Select by Ruler: selects all text matching the paragraph style at the insertion point.

Select by Font Style/Ruler: selects all text matching the paragraph style and font style at the insertion point.

Select by Font Family/Ruler: selects all text matching the paragraph style and font family at the insertion point (italic, bold, etc. font variants are included in the selection).

Match

All/Selection to Default Style: matches the selected text (or the whole document if there is no selection) to the default style as defined in the Styles pane of Preferences.

All/Selection to Most Common Style: matches the selected text to the most common font style and paragraph style found in that selection; if there is no selection, all styles in the document will be ‘simplified’ or ‘flattened’ to the most common font style and paragraph style.

All/Selection to Style at Text Cursor: propagates the font style and paragraph style found at the insertion point throughout selected text, or throughout the whole document if there is no selection.

Remove All Styles: removes all font styles and paragraph styles throughout the entire document, leaving OS X’s default style (Helvetica 12, etc.). Also removes all text tables, list attributes, and attachments. 

The above command is equivalent to the Make Plain Text followed by Make Rich Text sequence in Text Edit, which people often use to strip web text of all styles and attachments.
