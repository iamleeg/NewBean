Shortcut Keys


There are shortcut key combinations in the menus for many Bean-specific commands.


In addition, here are some standard OS X text-editing keyboard shortcuts. Some of them are not intuitively obvious, especially for people with a Windows PC background.


⌃⌘D  ...  popup dictionary (courtesy OS X)

[esc]  ...  word completion hints (courtesy OS X)

⌘↑  ...  cursor to beginning of document

⌘↓  ...  cursor to end of document / force completed layout

⌥←  ...  cursor to word left

⌥→  ...  cursor to word right

⌃↑  ...  view page up

⌃↓  ...  view page down

fn↑  ...  scroll view page up

fn↓  ...  scroll view page down

fn⌥↑  ...  cursor page up

fn⌥↓  ...  cursor page down

⇧↑  ...  select line up

⇧↓  ...  select line down

fn⇧↑  ...  select page up

fn⇧↓  ...  select page down

⇧⌘↑  ...  select to beginning of document

⇧⌘↓  ...  select to end of document

⌥⌫  ...  erase word left

fn⌥⌫  ...  erase word right

⌥[  ...  force straight double quote (when 'Smart Quotes' is on)

⌥]  ...  force straight single quote (when 'Smart Quotes' is on)

⌃[return]  ...  insert line break (Leopard only)

⌥[space bar]  ...  insert non-breaking space

[scroller knob]⌥  ...  extra slow scroll (Leopard only)

[ctrl]t  ...  transpose the two characters nearest to the insertion point


For Lists:


⌥[tab]  ...  create new list

[tab]  ...  indent list item

⇧[tab]  ...  un-indent list item

⌃⌥[tab]  ...  insert tab into list item (OS X 10.5+)

⌥⌘→  ...  indent list item (in Bean only)

⌥⌘←  ...  un-indent list item (in Bean only)

NOTE: [return] on an empty line un-indents a new list item


Find Panel:


⌥[tab]  ...  insert tab character into search field

⌥[return]  ...  insert return character into search field

[option]  ...  causes Replace All/Select All (in entire document) to become Replace All/Select All (in selection only)


Color Panel:


⇧[option]  ...  will set the highlight color (background color) of selected text instead of the font color (in Bean only)


Symbols Legend:


⌃ control key

⌘ command  key

⌥ option (or alt)

fn function key

⇧ shift key

⌫ delete key
